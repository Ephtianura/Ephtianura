<h2 align="left">🌸 Best project: AniFlow</h2>

> **[AniFlow.xyz](https://aniflow.xyz)** – агрегатор аніме-контенту з фокусом на українську аудиторію.

Проєкт з’явився з ідеї зробити доступ до аніме більш зручним для українських користувачів. 

Основний фокус зроблений на надійному бекенді та якісному UX, разом з адаптацією під різні пристрої.

<!-- <img src="https://skillicons.dev/icons?i=redis" alt="" width="15" /> -->
Реалізовано кешування як власного контенту, так і зовнішніх плеєрів.
Для зберігання контенту та можливості масштабування використовується S3, а для деплою – Nginx як reverse proxy.

Окремо продумана інтеграція з кількома зовнішніми відеоплеєрами та синхронізація контенту у власну базу даних для стабільного оновлення каталогу.

Проєкт все ще розвивається і поступово обростає новими можливостями.
